🏛️ 8부 통합 — 자라라 센터
· 최종 금고 실제 입력코드 = 91  (체크섬: 7부스 답의 모든 자릿수 합 8+19+15+13+19+2+15 = 91)
· 도장판 '상생 지수' = 22  (의미수: 각 답 첫째자리 합 3+3+2+5+4+2+3 = 22) — 표시 전용, 입력하지 않음
🚨 가장 중요: 체크섬 91과 의미수 22는 완전 독립. 절대 산술로 연결 금지 (M-034 AGG-5). 학생이 22를 금고에 넣으려 하면 "그건 화면에 보여주는 상생 지수예요. 금고엔 정산판 검증값(91)을." 으로 안내.

② 시간 배분

총 시간은 회차별 현장 측정(미검증). 권장: 부스당 8~12분 + 8부 통합 10분 + 도입/디브리핑. 부스 순회식이므로 팀별 시차 운영 가능.

③ 도입 스크립트

"여러분은 광명의 상생을 회복하는 소셜이코노미 헌터즈입니다. 8개 현장이 위기에 빠졌어요. 각 현장의 문제를 수학으로 풀면 스탬프를 얻고, 마지막 자라라 센터에서 흩어진 답이 하나로 모입니다. 혼자선 못 풀어요 — 함께라야 열립니다. 자, 첫 번째 동굴로!"

④ 개입 전략·힌트 타이밍

막힌 지 ~5분 후 ①개념 힌트, +3분 후 ②절차 힌트, 종료 임박 시에만 ③직접 제시. 힌트 카드(02)를 봉투에서 단계별로.

⑤ 부스별 문제 상황 대응

부스운영 메모 · 대응
1부· UV펜은 연출(답 생성 필수 아님). 틀려도 물리 신호 없음 → '다시 천천히 세어볼까요?' 유도.
· 협력·연대 둘 다 3번이라 헷갈림 → 순서(협력 먼저) 강조.
2부· 강응결: 비례값=저울 균형점. 틀리면 저울 기움. 암산만 하려 하면 저울 사용 권유.
· ③ 양파는 답이 커지는 문제(9kg). 소요 8~12분(미검증).
3부· 최상 응결: 비추는 몸짓이 곧 답. 머리로 뒤집으려는 학생 많음 → '직접 거울에.'
· ⚠ 인쇄 전 실제 거울로 2058 가독성 검수. 거울 청결·조명 충분.
4부· 중응결·선회: 걷기→긋기로 후퇴. 경로 맞는데 덧셈 틀리는 경우 많음 → 덧셈 함께 확인.
· 역 누락·중복 → 지나간 역 표시하며 진행.
5부· 제거형(8부스 유일): 가짜를 걷어냄. 가짜 카드는 '권한 없는 주장'(정당한 오도).
· 다 더하려 하면 → 규칙 현판으로 되돌림. 틀려도 물리 신호 없음 → '규칙 다시 볼까요?'
6부· 강응결·수직결합: 부피→도시구원. 구현의존 높음 → 모래·탱크·게이지 점검 중요.
· π 쓰려는 학생엔 '밑면적이 이미 주어졌어요.' 한 나무 빼고 더하기 주의.
7부· 강응결·climactic: 추론한 순서=엮은 배열=코드. 추상도 높음 → 막히면 시작점(양전) 짚기.
· 선후 못 잡으면 '땅 조사→등급?' 인과 유도. decoy 목간 1~2점은 후반 긴장용.

⑥ 디지털 백업

교구 고장 시: 각 부스 문항을 종이/화면으로 대체 제시. 거울(3부)·저울(2부)·모래탱크(6부)는 물리 응결이 핵심이라 가급적 실물 유지. 부득이하면 답만 확인하는 방식으로 운영(응결 등급 하락 감수).

⑦ 안전 점검

거울 파손·UV펜 눈 직사 주의·저울 추 낙하·모래 흩날림 주의. 부스 동선에 걸림 없는지 사전 확인. 상세는 18 안전 점검 일지.
